# Login page 

## Overview

Login page UI layout. Use the screenshot referenced below for how it should look.

## Requirements

- Add reactive form with email and password as formControls. Include email validation + required for the email control and required for the password control
- Add 'Email address' label and textbox wire up to formControl.
- Add 'Password' label and textbox wire up to formControl
- Email textbox should be filled #f2f5f7 and contain an email icon <i class="fa-sharp fa-light fa-envelope"></i> as well as placeholder: 'your@email.com'
- Password textbox should be filled #f2f5f7 and contain a padlock icon <i class="fa-light fa-lock"></i> as well as placeholder: 'Your password'
- Below the password textbox and aligned to the right add a router link to 'Forgot password?'
- Add a Log in button with background #0c2340
