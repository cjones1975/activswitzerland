# Footer navigation menu

## Overview

Navigation footer menu UI layout. Use the screenshot referenced below for how it should look.

## Requirements

- Design the footer as portrayed in the screenshot
- From left to right the icons must be the following:
Home - <i class="fa-light fa-house"></i>
Explore - <i class="fa-light fa-compass"></i>
Map - <i class="fa-light fa-map"></i>
Saved Trips - <i class="fa-light fa-bookmark"></i>
Profile - <i class="fa-sharp fa-light fa-circle-user"></i>
- An icon that is hovered over, must display slightly expanded with a blue circle around it. Like the map icon in the screenshot. The icon itself must be white
- Each icon will be a route to another component. By default, the Home icon which routes to the home component must be active
- An active route icon must ahve the color #b45309. All inactive route icons must have the color #0e3e6b
- Only the home route can be setup for now. Other routes will be done later.
